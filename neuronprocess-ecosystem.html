How to export as PDF for LinkedIn:
  1. Press Ctrl + P (or Cmd + P on Mac) to open the print dialog.
  2. Set Destination → "Save as PDF".
  3. Set Paper size → "Custom" → 800 × 800 px (or closest square option).
  4. Set Margins → "None".
  5. Enable "Background graphics" in More settings.
  6. Click Save. Then upload the PDF directly to a LinkedIn post — it will display as a carousel.
